# Movie Ticket Reservation System Using Python, Flask, HTML & CSS

A web-based movie ticket reservation system developed using Python Flask for the backend and HTML/CSS for the frontend. This application allows users to browse movies, select show timings, book seats, and manage reservations through an interactive and user-friendly interface.

## Features

* User-friendly movie booking interface
* Display available movies and show timings
* Seat selection and reservation system
* Booking confirmation page
* Responsive frontend design
* Dynamic data handling using Flask
* Simple and secure reservation workflow

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* Bootstrap
* SQLite (optional database)
* Jinja2 Templates

## Modules Included

* Home Page
* Movie Listings
* Show Timing Selection
* Seat Booking
* Reservation Confirmation
* Admin Panel (optional)

## Project Workflow

1. User visits the homepage
2. Selects a movie and show timing
3. Chooses available seats
4. Confirms booking
5. Reservation details are stored in the database
6. Booking confirmation is displayed

## Key Functionalities

* Real-time seat availability
* Form validation
* Database integration for storing bookings
* Dynamic webpage rendering using Flask templates
* Clean and responsive UI design

## Database Tables

### Movies

* Movie ID
* Movie Name
* Genre
* Duration
* Show Timing

### Reservations

* Reservation ID
* User Name
* Selected Seats
* Booking Date
* Payment Status

## How to Run the Project

1. Clone the repository
2. Install required dependencies:

   ```bash
   pip install flask
   ```
3. Run the Flask application:

   ```bash
   python app.py
   ```
4. Open the browser and visit:

   ```bash
   http://127.0.0.1:5000/
   ```
